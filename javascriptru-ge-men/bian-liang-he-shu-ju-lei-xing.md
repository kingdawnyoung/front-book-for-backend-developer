### 变量

变量是对“值”的引用，使用变量等同于引用一个值。每一个变量都有一个变量名。

js里三种定义变量方法， var, let，const

#### var

```js
var foo;
var bar = 1;
console.log(foo, bar);//undefined, 1
```

#### let

```js
let foo;
let bar = 1;
console.log(foo, bar);//undefined, 1
```

#### const

```
const foo;
const bar = 1;
console.log(foo, bar);//undefined, 1
```
